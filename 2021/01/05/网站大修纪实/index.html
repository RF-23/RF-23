网站大修纪实

网站大修纪实

一月 05, 2021
  • 杂项

环境：

Hexo + diaspora + github + coding

起因：

起因当然很简单咯，因为旧的崩了…还有成吨的bug…

问题

  • 没有做mathjax支持，导致latex公式全都渲染不出来
  • 受疫情影响github服务器渐渐土豆化
  • 受没有充钱影响码云服务器开始出bug
  • 没有移动端支持，手机访问体验极差

    解决方法

  • 参照此方法添加了mathjax支持，且主题更新后也添加了mathjax支持
  • 疫情过后github速度有所提升，且我同时使用了coding的pages服务，将此域名同时指向两个ip(在解析中添加多个A类地址)
  • 放弃码云
  • 主题更新后增加了移动端支持，因为手机竖屏显示的封面图我也更换了封面图,使得构图更加和谐。据说图太丑了，但我也没什么办法，有好图再换吧。

    新增功能

  • 搜索：封面的右上角放大镜图标，可搜索站内文章及文章内容
  • 每天一个致郁小技巧：封面右上角资料库图标，提供致郁小句子…
  • 右下角的板娘：添加了live2d插件

    音乐库

    点击左上角关闭音乐，没有正常加载刷新网页即可

缺陷

  • latex公式必须手动刷新网页才能被正确的渲染
  • 有些手机没有陀螺仪，导致封面动态图的交互效果失效
